# Instrucciones para generar el archivo PPTX

He preparado la versión web (reveal.html) y las notas (slides_notes.md).

Opciones para obtener el PPTX:

1) Crear el PPTX localmente (PowerPoint / Google Slides):
   - Abre PowerPoint o Google Slides.
   - Usa una plantilla blanca con acentos azules (#0b3d91 y #2b7cff).
   - Crea 9 diapositivas siguiendo slides_notes.md; inserta las imágenes sugeridas.
   - Añade las notas del presentador (usa el contenido de slides_notes.md).
   - Exporta como .pptx.

2) Subir los archivos al repositorio:
   a) En GitHub, si no existe rama main, crea un README.md (Add file → Create new file → Commit) — esto crea la rama principal.
   b) Sube reveal.html, slides_notes.md y pptx_instructions.md a la rama main (Add file → Create new file o Upload files).
   c) (Opcional) En Settings → Pages elige Branch: main y carpeta root, guarda y la URL pública será:
      https://Niki-0311.github.io/Presentacion-sociolinguistica/reveal.html

3) Si prefieres que yo genere el .pptx y lo suba por ti, indícamelo después de que confirmes que quieres que lo haga; cuando me autorices a subir, lo crearé y lo subiré al repo in a following step.
